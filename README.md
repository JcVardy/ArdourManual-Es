This is a clone of the Ardour manual meant to be used to work on the spanish translation.

Esta es un réplica del manual de Ardour creada y pensada para trabajar en la traducción del programa al español.

La carpeta en inglés (ardour-manual) está enlazada al repositorio del manual oficial de Ardour como referencia para el trabajo.

La carpeta en español (ardour-manual-esp) contiene un archivo de texto (SpanishMasterDocument-DocumentoMaestroEspañol) dónde podemos observar los criterios de la traducción.

El propósito de este repositorio es que, ya que el programa y su documentación son de código abierto, podamos ahunar criterios para lograr la mejor traducción posible.
Mientras más ojos haya puestos en la traducción, mejor.
