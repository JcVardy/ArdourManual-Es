El contenido de este cuadro de opciones depende parcialmente del tipo de datos en el clip, ya sea audio o MIDI. Las opciones comunes son:

  • Nombre: el nombre completo del archivo del clip, usualmente visible en forma abreviada.
  • Cargar: este botón permite cargar el archivo del clip en la ranura de activación seleccionada.
  • Color: abre un selector de color para establecer el color del título del clip dentro de la ranura de activación.

La única opción específica de audio es Ganancia (dB), que ajusta el volumen de salida de un clip.

Para los clips MIDI, hay varias opciones específicas:

  • Ajuste de Velocidad: ajusta efectivamente el volumen de salida incrementando la velocidad de las notas MIDI.
  • Enviar Parches: habilita el envío de un mensaje de control para cargar un parche específico. Esto se usa comúnmente con equipos MIDI externos como sintetizadores de hardware.
  • Parches MIDI: al hacer clic en este botón se abre un diálogo donde se puede seleccionar un parche para enviar cuando se active un clip MIDI.
