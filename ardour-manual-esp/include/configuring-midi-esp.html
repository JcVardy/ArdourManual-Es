MIDI es una forma de describir interpretaciones musicales y de controlar hardware y software musical.

Ardour puede importar y grabar datos MIDI, y realizar una amplia variedad de operaciones de edición sobre ellos. Además, el protocolo MIDI puede utilizarse para controlar diversas funciones de Ardour.

Infraestructuras de gestión MIDI

La entrada y salida MIDI en Ardour se gestionan internamente mediante el mismo «motor» que maneja la entrada y salida de audio. Sin embargo, Ardour puede utilizar tantos dispositivos MIDI como el sistema pueda reconocer, ya que no existen dificultades de sincronización como las que se dan con el audio.

OS X CoreMIDI es la infraestructura MIDI estándar en sistemas OS X.
Linux ALSA MIDI es la infraestructura MIDI estándar en sistemas Linux.
Windows MME es la infraestructura MIDI estándar en sistemas Windows.

JACK es un sistema de audio alternativo que Ardour puede utilizar tanto para audio como para MIDI. JACK se usa para enrutar audio entre aplicaciones independientes y hoy en día se considera un caso de uso avanzado que no se recomienda para la mayoría de los usuarios. Los usuarios que necesiten utilizar JACK para el ruteo de audio deberían consultar la documentación más reciente en el sitio web de JACK.
